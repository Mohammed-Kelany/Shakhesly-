# شخصلي AI - Shakhesly AI

منصة ذكية لاكتشاف أعطال الأجهزة المنزلية والمركبات وإدارتها باستخدام الذكاء الاصطناعي.  
تربط المنصة بين العملاء والفنيين المعتمدين من خلال نظام عروض تنافسية، مع تذكير بالصيانة الدورية ونظام إشعارات متكامل.

---

## 🚀 المميزات الرئيسية

- **🤖 تشخيص ذكي بالذكاء الاصطناعي** (يدعم ربط OpenAI للحصول على تشخيص احترافي فوري).
- **📱 سلايدر تفاعلي** لعرض مختلف الأجهزة والمركبات المدعومة.
- **🔧 نظام العروض التنافسية**: أكثر من فني يقدم عرض سعر، والعميل يختار الأنسب.
- **💬 شات مباشر** بين العميل والفني بعد قبول الطلب.
- **📅 تذكير تلقائي** بمواعيد الصيانة الدورية لكل جهاز مسجل.
- **🔔 إشعارات فورية** للطلبات الجديدة وتحديثات الحالة.
- **📸 رفع صور الأعطال** عبر الكاميرا أو المعرض.
- **📱 PWA** – يمكن تثبيت المنصة على الهاتف واستخدامها كتطبيق.
- **🌐 دعم كامل للغة العربية** واتجاه RTL.

---

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|--------|-----------|
| HTML5, CSS3, JavaScript | بناء الواجهة الأمامية (Vanilla JS) |
| Firebase Realtime Database | تخزين البيانات (مستخدمين، أجهزة، طلبات) |
| OpenAI API | التشخيص الذكي الاختياري |
| CryptoJS | تشفير كلمات المرور |
| Font Awesome 6 | الأيقونات |
| Google Fonts (Cairo) | الخط العربي |
| AOS (Animate on Scroll) | تأثيرات الحركة والظهور |
